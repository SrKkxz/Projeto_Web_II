# Zenith

Sistema web gamificado de desafios pessoais e conquistas desenvolvido com Django.

---

## Descrição

O Zenith é uma plataforma de desafios pessoais onde usuários podem registrar metas, concluir missões e ganhar experiência (XP) conforme avançam em seus objetivos.

O sistema utiliza gamificação para incentivar produtividade, organização e evolução pessoal.

## Como Executar o Projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/SrKkxz/Projeto_Web_II.git
```

---

### 2. Entrar na pasta do projeto

```bash
cd zenith
```

---

### 3. Criar ambiente virtual

```bash
python -m venv venv
```

---

### 4. Ativar ambiente virtual


```bash
.\venv\Scripts\activate
```

---

### 5. Instalar dependências

```bash
pip install django
```

---

### 6. Rodar servidor

```bash
python manage.py runserver
```
